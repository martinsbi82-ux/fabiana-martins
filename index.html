My Approach

The Commercial Reset Framework™

A structured 4-phase method applied across every retainer engagement. Each phase builds on the previous — so results compound over time instead of resetting every month.

Phase 01
Diagnose
Map the current commercial operation, identify bottlenecks, and establish a baseline of where deals are being lost.
Commercial process audit
Funnel leak analysis
Priority gap matrix
Phase 02
Design
Build the commercial architecture: pipeline stages, roles, criteria, and documentation your team will actually use.
Pipeline structure & stage definitions
CRM configuration guide
Commercial playbook (v1)
Phase 03
Deploy
Implement the new system with the team, run onboarding, and establish weekly rhythms that sustain the process.

KPI dashboard setup
